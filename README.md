# azure-apim-rbac-implementation
The "azure-apim-rbac-implementation" repository provides scripts and documentation for implementing RBAC in Azure API Management. It offers detailed access control at API and operation levels, includes lock management, and integrates with CI/CD pipelines. This solution enhances API security and streamlines access management in Azure.
For detailed implementation, check blog post- at https://ramathotapalli.hashnode.dev/evaluating-azure-apim-workspace-isolation-is-the-50k-investment-worth-it
