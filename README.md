# azure-apim-rbac-implementation
The azure-apim-rbac-implementation repository provides scripts and documentation for implementing RBAC in Azure API Management. It offers detailed access control at API and operation levels and integrates seamlessly with CI/CD pipelines. This solution enhances API security and streamlines access management in Azure.

For detailed implementation, check the blog post at https://ramathotapalli.hashnode.dev/implementing-granular-rbac-in-azure-api-management-a-cost-effective-alternative-to-workspaces
